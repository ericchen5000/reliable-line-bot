# Reliable AI Customer Service

An AI customer service prototype for LINE and web chat, built with FastAPI.

This repository contains the application code for a customer-support assistant, an admin dashboard, FAQ management, knowledge-base retrieval, website indexing, conversation logs, and account permissions.

## Features

- LINE chatbot integration
- Web chat API
- FAQ-based answers
- Knowledge-base search
- Website index search
- Conversation logs
- Admin dashboard
- Account and role management
- Basic image/OCR handling

## Tech Stack

- Python
- FastAPI
- Uvicorn
- LINE Bot SDK
- DeepSeek-compatible chat API

## Configuration

Runtime configuration is handled through environment variables.

Do not commit `.env`, API keys, tokens, logs, user data, or production-only files.

## Notes

This is an application project, not a public SDK. Deployment details, production credentials, internal URLs, operational notes, and customer data are intentionally not included in this public README.

Internal maintenance documentation should be kept outside the public repository or in an ignored internal document.
